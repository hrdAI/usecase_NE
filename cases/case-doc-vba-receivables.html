📌 VBA (Visual Basic for Applications)란?

Excel 내에서 반복 작업을 자동화하는 프로그래밍 언어예요. SAP에서 다운로드한 원시 데이터를 수동으로 가공하고, 다른 파일의 값을 찾아 맵핑하는 복잡한 집계 과정을 버튼 하나로 처리할 수 있게 해줍니다.

1. 성과요약

VBA 자동화를 통해 월 3~4시간 소요되던 수작업 데이터 가공 시간을 월 0.5시간으로 단축시켜, 연간 30시간 이상의 업무 시간을 절약하는 성과를 달성했습니다.

➡️ 연간 30시간 이상 업무 시간 절약!

📅 소요 시간:

월 3~4시간 (연간 약 42시간)

👥 작업 방식:

  • SAP 데이터 다운로드
  • 사업부별 Sheet 가공 작업 (수작업)
  • 다른 파일의 값을 찾아 수작업 맵핑
  • 보고용 사업부 요약 집계 작업 (수작업)

⚠️ 문제점:

  • 매월 3~4시간 반복 작업
  • 수작업 맵핑으로 인한 휴먼 에러
  • 복잡한 집계 과정
  • 담당자 숙련도에 따른 품질 차이

월 0.5시간 (연간 약 6시간)

  • 사업부별 Sheet 자동 가공
  • 타 파일 데이터 자동 맵핑
  • 보고용 사업부 요약 집계 자동 생성

✅ 개선 효과:

[ 비효율 ]

📌 Google Apps Script (GAS)란?

Google 스프레드시트, 문서 등 Google 서비스와 연동하여 작업을 자동화하는 스크립트 언어예요. 웹페이지 정보를 자동으로 가져오고(크롤링), 정해진 시간마다 실행(트리거)할 수 있게 해줍니다.

1. 성과요약

Google Apps Script(GAS) 자동화를 통해 월 2시간 소요되던 작업을 월 0.02시간으로 단축시켜, 연간 24시간 이상의 업무 시간을 절약하는 성과를 달성했습니다.

➡️ 연간 24시간 이상 업무 시간 절약!
자동화 이전

📅 소요 시간:

월 2시간 (주1회, 1회당 30분)

👥 작업 방식:

  • 예스24에서 교재명 수동 검색
  • 판매지수 수동 복사
  • 엑셀/구글시트에 수동 입력
  • 주1회 반복 작업

⚠️ 문제점:

  • 휴먼 에러 발생
  • 담당자 부재 시 데이터 공백
  • 교재 수 증가 시 부담 증가
자동화 이후

⚡ 소요 시간:

월 0.02시간

🤖 작업 방식:

  • GAS 스크립트 자동 실행
  • YES24 자동 크롤링
  • 판매지수 자동 추출
  • 구글시트 자동 기록
  • 트리거로 주1회 자동 실행

✅ 개선 효과:

  • 연간 24시간 절약
  • 휴먼 에러 제거
  • 담당자 없어도 자동 실행
  • 교재 수 무관하게 동일 시간

2. 개발배경

[ 업무 상황 ]
주 1회 온라인서점 '예스24'에서 자사 제품 및 경쟁서의 판매 지수를 엑셀에 기록해 왔습니다. (판매 추이 및 시장 동향 파악 목적)

[ 비효율 ]
수기로 진행하다 보니 최소 30분 이상 소요되고, 휴먼 에러 가능성이 있었습니다. 또한 담당자 부재시에는 기록 공백이 생겨 안정적으로 데이터를 축적하기 어려웠습니다.

[ 확장 문제 ]
현재 팀에서 관리하고 있는 교재 권수는 40여 권이 넘으며, 앞으로 출간 교재 종수가 늘어날수록 관리해야 할 책은 많아지는 상황이었습니다.

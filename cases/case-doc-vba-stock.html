📌 VBA (Visual Basic for Applications)란?

Excel 내에서 반복 작업을 자동화하는 프로그래밍 언어예요. SAP 등에서 다운로드한 원시 데이터(로우 데이터)를 사람이 수동으로 정리/분류/매칭하는 복잡한 과정을 버튼 하나로 처리할 수 있게 해줍니다.

1. 성과요약

VBA 자동화를 통해 월 3시간 소요되던 반복적인 재고 집계 작업을 월 0.5시간으로 단축시켜, 연간 30시간 이상의 업무 시간을 절약하는 성과를 달성했습니다.

➡️ 연간 30시간 이상 업무 시간 절약!
자동화 이전

📅 소요 시간:

월 3시간 (연간 약 36시간)

👥 작업 방식:

  • SAP 로우 데이터 다운로드
  • 불필요한 창고 제거 및 합계 없는 행 삭제 (수작업)
  • 사업군별 창고 시트 분류 (수작업)
  • 지정되지 않은 창고를 사업군별 시트에 매칭 (수작업)
  • 팀별/사업별 요약 보고서 작성 (시트별 수식 확인)

⚠️ 문제점:

  • 반복적인 수작업으로 평균 3시간 소요
  • 데이터 분류 및 매칭 과정의 휴먼 에러
  • 담당자마다 결과물 품질 차이
  • 시트 분류 작업의 반복성
자동화 이후

⚡ 소요 시간:

월 0.5시간 (연간 약 6시간)

🤖 작업 방식:

  • SAP 로우 데이터 다운로드 (붙여넣기)
  • VBA 매크로 실행 버튼 클릭
  • 불필요 행/창고 자동 제거
  • 사업군/창고별 시트 자동 분류
  • 팀별/사업별 요약 보고서 자동 생성

✅ 개선 효과:

  • 연간 30시간 이상 절약
  • 데이터 분류 및 매칭 자동화
  • 휴먼 에러 제거

[ 업무 상황 ]
매월 사업군·창고·품목 단위로 재고현황을 집계해야 했습니다.

[ 비효율 ]
SAP에서 로우 데이터를 다운받아 불필요한 데이터를 제거하고, 시트를 분류하며, 창고를 매칭하는 반복적인 수작업으로 인해 평균 3시간이 소요되었습니다.
